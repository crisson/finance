How do I get it?

Start by downloading the production or minified version of the library. Include a reference to the library in a script tag inserted at the top or bottom of your HTML document.

<script src="js/finance.min.js"></script>

The minified library includes all required dependencies. If you're skipping the minified version, you'll have to include both finance.inputs.js and finance.js

<script src="js/finance.inputs.js"></script>
<script src="js/finance.js"></script>
